# `@glass/ui`

Shared UI primitives and design tokens for Glass. Static HTML prototypes live in `/REFERENCES` at the repo root; see `DESIGN_NOTES.md` for how they map into this package.

## Consumption

Import tokens in an app entry or layout:

```css
@import '@glass/ui/tokens.css';
```

```tsx
import { Button } from '@glass/ui'
```

See `DESIGN_NOTES.md` for how tokens map to the HTML references.
